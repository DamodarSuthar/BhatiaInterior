Download the Android Native Application

While our web simulation offers exceptional client feedback, testing on real physical screens is unmatched. Our application compiles directly into a standalone Debug APK built on modern architectures:

Native Architecture Specs

  • Full support across Android SDK 26 (Android 8.0) up to Android 14 (SDK 34).
  • Optimized Jetpack Compose vector layouts for instantaneous responsiveness.
  • Edge-to-edge system bar layout with full gesture navigation awareness.
  • Room SQLite Database local persistence.

Local Gradle Compilation

If you clone this repository to your local system, you can build and package the native application instantly using the standard Gradle wrappers:

Terminal Commands Android Studio CLI
# Navigate to the project root directory
cd bhatia-interior
# Clean previous builds (optional)
./gradlew clean
# Compile and assemble debug APK
./gradlew assembleDebug
# Output folder: app/build/outputs/apk/debug/app-debug.apk
